# NeoNote

NeoNote is a powerful Chrome extension designed to help you highlight, take notes, and organize information directly within your browser. It features AI-powered summarization, seamless synchronization with Lark (Feishu) Bitable, and a clean, efficient sidebar interface.

## üöÄ Features

-   **Sidepanel Note-taking**: A dedicated sidebar to jot down thoughts without leaving your current tab.
-   **Web Highlighting**: Highlight text on any webpage; it's automatically saved as a note with a link back to the source.
-   **AI Summarization**: Use advanced LLMs to summarize page content or generate insights from your notes.
-   **Lark Synchronization**: Sync your notes directly to a Lark (Feishu) Bitable for better organization and collaboration.
-   **Markdown Support**: Basic markdown support for lists and formatting.
-   **Dark Mode**: Comfortable UI that adapts to your system theme.
-   **Multilingual**: Support for both English and Chinese.

## üõ† Installation (Developer Mode)

1.  Clone this repository to your local machine.
2.  Open Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked** and select the folder containing this project.

## ‚öôÔ∏è Configuration

### Lark Synchronization
1.  Open the NeoNote Home page (via the home icon in the sidebar).
2.  Navigate to **Settings**.
3.  Enter your Lark Bitable link and Personal Access Token.

### AI Summarization
1.  In the **Settings** page, provide your LLM API Key and Base URL.
2.  Choose your preferred model and enable/disable "Thinking" mode as needed.

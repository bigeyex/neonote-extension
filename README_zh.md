# NeoNote

NeoNote 是一款功能强大的 Chrome 扩展程序，旨在帮助您直接在浏览器中进行高亮、记录和整理信息。它具有 AI 摘要、与飞书多维表格（Lark Bitable）无缝同步以及简洁高效的侧边栏界面。

## 🚀 功能特性

-   **侧边栏笔记**：专用的侧边栏，让您在不离开当前页面的情况下记录想法。
-   **网页高亮**：在任何网页上高亮文本；它会自动保存为笔记，并附带指向源内容的链接。
-   **AI 摘要**：利用先进的大语言模型（LLM）总结页面内容或从您的笔记中生成洞察。
-   **飞书同步**：将您的笔记直接同步到飞书多维表格，以便更好地组织和协作。
-   **Markdown 支持**：支持列表和格式化的基础 Markdown 语法。
-   **深色模式**：适应系统主题的舒适 UI。
-   **多语言**：支持中英文。

## ⌨️ 快捷键

| 操作 | 快捷键 (Mac) | 快捷键 (Windows/Linux) |
| :--- | :--- | :--- |
| **切换侧边栏** | `Command + 0` | `Ctrl + 0` |
| **将选定内容添加为笔记** | `Command + Shift + 0` | `Ctrl + Shift + 0` |
| **保存笔记 (编辑器)** | `Command + Enter` | `Ctrl + Enter` |

## 🛠 安装方法 (开发者模式)

1.  克隆此仓库到本地。
2.  打开 Chrome 浏览器，访问 `chrome://extensions/`。
3.  在右上角启用 **开发者模式**。
4.  点击 **加载已解压的扩展程序**，然后选择包含此项目的文件夹。

## ⚙️ 配置说明

### 飞书同步
1.  打开 NeoNote 主页（通过侧边栏中的主页图标）。
2.  进入 **设置** 页面。
3.  输入您的飞书多维表格链接和个人访问令牌（Personal Access Token）。

### AI 摘要
1.  在 **设置** 页面中，提供您的 LLM API Key 和 Base URL。
2.  选择您喜欢的设计模型，并根据需要启用/禁用“思考（Thinking）”模式。
